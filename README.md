# 📝 To-Do App

A simple and responsive To-Do App built using **HTML, CSS, and JavaScript**.  
This application helps users manage their daily tasks efficiently by allowing them to add and delete tasks easily.

## 🎯 How It Works

1. Enter a task in the input field
2. Click the **ADD Task** button
3. Your task will appear in the task list
4. Click the **Delete** button to remove a task
